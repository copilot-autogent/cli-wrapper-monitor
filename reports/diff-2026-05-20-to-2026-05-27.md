## Root Cause Analysis

### 🟡 Tool definitions grew +9.3%

**New tool added**: `pipeline_add` (~700 chars) — routes observation task findings into the work-pipeline memory topic. This is a 1-tool addition; not concerning.

### ⚪ System prompt grew +1.4%

**Cause**: USER.md grew from 11,872 → 13,216 chars (+1,344, +11.3%). USER.md is within the 20k truncation limit so the full growth landed in the prompt. PLAYBOOK.md and CONTEXT.md are still truncated — their growth is silently dropped.

### ⚠️ Bootstrap truncation: still active

PLAYBOOK.md has grown further since May 20 (39,673 → **53,583** chars, +35%). CONTEXT.md grew from 41,739 → **47,057** chars (+12.7%). Both files are growing rapidly. The truncation cut is now deeper than before.

Fix is pending in **PR #383** ([JackywithaWhiteDog/autogent#383](https://github.com/JackywithaWhiteDog/autogent/pull/383)): raises `maxCharsPerFile` default from 20k to 60k. When merged, the next baseline will show a large positive "regression" in system prompt chars (+105% projected) — this is expected and reflects the fix delivering the full content.
